[2 Com-BIFSOPA Heading 1] SECTION 181 – Code of conduct for adjudicators 
[1 BIFSOPA Heading] 181    Code of conduct for adjudicators 
[1.3 BIFSOPA level 1 (CDI)] The registrar may make a code of conduct for adjudicators. 
[1.3 BIFSOPA level 1 (CDI)] The code of conduct, or an amendment or replacement of the code, does not take effect until approved by regulation. 
[1.3 BIFSOPA level 1 (CDI)] The Minister must, within 14 sitting days after the code of conduct, or an amendment or replacement of the code, takes effect, table a copy of the code in the Legislative Assembly.
[1.3 BIFSOPA level 1 (CDI)] A failure to comply with subsection (3) does not invalidate or otherwise affect the code of conduct.
[1.3 BIFSOPA level 1 (CDI)] The commissioner must— 
[1.4 BIFSOPA level 2 (CDI)] publish the code of conduct on the commission’s website; and 
[1.4 BIFSOPA level 2 (CDI)] keep copies of the code of conduct available for inspection, without charge, at the commission’s office at any time that office is open to the public; and
[1.4 BIFSOPA level 2 (CDI)] if asked, advise where copies of the code of conduct may be obtained.
